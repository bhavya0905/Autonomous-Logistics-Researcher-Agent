# Autonomous Logistics Researcher Agent

## Overview
An AI-powered research system that automatically investigates a query, gathers relevant information from the web, and generates a structured report with supporting evidence.

The project combines web search, scraping, analysis, and report generation using Large Language Models (LLMs) to streamline the research workflow.

---

## Features
- Automated web search and data collection
- Intelligent scraping and information extraction
- AI-powered analysis using LLMs
- Structured report generation
- Evaluation metrics for:
  - Quality
  - Relevance
  - Coverage
- Modular and scalable architecture

---

## Tech Stack
- **Python**
- **Streamlit**
- **Large Language Models (LLMs)**
- **Vector Databases**
  - ChromaDB
  - FAISS
